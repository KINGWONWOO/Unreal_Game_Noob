# NoobGame (Project Portfolio)

**NoobGame**은 언리얼 엔진 5(Unreal Engine 5)를 기반으로 제작된 멀티플레이어 파티 게임 프로젝트입니다. 다양한 미니게임을 통해 플레이어들이 경쟁하고 협력할 수 있는 환경을 제공하며, 로비 시스템과 다채로운 게임 모드를 포함하고 있습니다.

---

## 🎮 주요 기능 및 게임 모드

### 1. 로비 시스템 (Lobby System)
- 플레이어들이 게임 시작 전 대기하고 상호작용할 수 있는 공간입니다.
- 캐릭터 선택 및 커스터마이징 기능 지원 (고양이, 강아지, 라쿤 등 다양한 캐릭터).
- 직관적인 UI를 통한 게임 모드 선택 및 매칭 시스템.

### 2. 과일 게임 (Fruit Game)
- **개요:** 제한된 시간 내에 정해진 과일을 수집하거나 상호작용하는 미니게임입니다.
- **특징:**
  - 물리 엔진을 활용한 과일 오브젝트 상호작용.
  - 드래그 앤 드롭 방식의 UI/UX 구현.
  - 점수 집계 및 순위 시스템.

### 3. 미로 게임 (Maze Game)
- **개요:** 복잡한 미로를 탈출하는 서바이벌 모드입니다.
- **특징:**
  - 다양한 테마의 맵 (백룸, 거울 미로 등).
  - 긴장감을 조성하는 조명 및 사운드 효과.
  - 길 찾기 알고리즘 및 트랩 요소 구현.

### 4. 퀴즈 게임 (Quiz Game)
- **개요:** O/X 퀴즈를 통해 생존자를 가리는 서바이벌 모드입니다.
- **특징:**
  - 실시간으로 정답/오답 구역으로 이동하는 액션 요소.
  - 탈락자 처리를 위한 데드존(DeadZone) 및 관전 모드.
  - 다양한 문제 데이터베이스 연동.

### 5. 공포 요소 (Horror Elements)
- **개요:** 일부 모드에 적용된 공포 분위기 연출 및 시스템입니다.
- **특징:**
  - 1인칭/3인칭 시점 변환.
  - 손전등, 야간 투시경 등 아이템 상호작용.
  - 심장박동 소리, 깜짝 놀래키는 요소(Jump Scare) 등의 연출.

## 🛠️ 기술 스택 (Tech Stack)

- **Engine:** Unreal Engine 5
- **Language:** Blueprints, C++
- **Version Control:** Git, GitHub
- **Platform:** PC (Windows)

## 📂 프로젝트 구조

```
Content/
├── Blueprints/       # 게임 로직 및 액터 설계
│   ├── FruitGame/    # 과일 게임 관련 블루프린트
│   ├── MazeGame/     # 미로 게임 관련 블루프린트
│   ├── QuizGame/     # 퀴즈 게임 관련 블루프린트
│   └── Lobby/        # 로비 시스템
├── Levels/           # 게임 맵 및 레벨 디자인
├── UI/               # 위젯 블루프린트 (HUD, 메뉴 등)
├── Characters/       # 플레이어 및 NPC 캐릭터 리소스
└── Sound/            # 효과음 및 배경음악
```
